# Troika! Character Sheet

A very simple browser-based character sheet for the [Troika!](https://www.troikarpg.com/) tabletop RPG. Go buy the game!

## Features

- Create and manage multiple characters
- Saves to browser LocalStorage
- Built in dice roller with roll-under skill testing
- Armour system that adjusts inventory capacity
- Import/export characters as JSON
- A somewhat reasonable responsive layout for mobile and desktop

## Usage

You can either use the [GitHub Pages version](https://akhtario.github.io/troika-character-sheet/) or download/clone the repository and open `index.html` in a browser.
